# Contributing to Kaida

Thank you for your interest in making AI agents safer.

## How to Help

**We need:**
- Bug reports from real-world usage
- Feature requests and ideas
- Testing on different operating systems (Windows, macOS, Linux)
- Additional policy templates for new agent types
- Documentation improvements

## Reporting Bugs

Open a [Bug Report](https://github.com/ajpandit775/kaida-shield/issues/new) on GitHub.

Include:
- OS and Python version
- Steps to reproduce
- Expected vs actual behavior
- Relevant log output from `~/.kaida/logs/`

## Feature Requests

Start a [Discussion](https://github.com/ajpandit775/kaida-shield/discussions) to propose new features or improvements.

## Security Vulnerabilities

**Do not open a public issue for security vulnerabilities.** See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Code of Conduct

Be respectful. Be constructive. We're all here to make AI agents safer.

Harassment, discrimination, and personal attacks will not be tolerated. Maintainers reserve the right to remove any contribution or contributor that violates these principles.
